# Portfolio

这是一个使用现代技术栈构建的个人作品集网站。

## 技术栈

### 前端
- React + TypeScript
- Vite
- Redux Toolkit

### 后端
- Node.js + TypeScript
- Express
- MongoDB

## 功能特性
- 用户认证
- 作品展示
- 响应式设计

## 部署
- 前端：静态部署
- 后端：Node.js 服务器
- 数据库：MongoDB

## 开发
```bash
# 安装依赖
cd frontend && npm install
cd backend && npm install

# 运行开发服务器
# 前端
cd frontend && npm run dev

# 后端
cd backend && npm run dev
```
